# 第一周：DP、Tree、Map核心算法

本周将深入学习算法题的三大核心领域：动态规划(DP)、树结构(Tree)和哈希表(Map)。每天专注一个主题，通过精选的LeetCode题目进行系统性练习。

## 本周学习目标
- 掌握动态规划的基本思想和经典模型
- 熟练处理各种树结构相关算法
- 灵活运用哈希表解决复杂问题
- 建立算法思维和解题模板

## 每日计划

### [第1天：动态规划基础](/plan/algorithm/week1/day1/)
**学习目标：** 掌握DP基本概念和一维DP问题
- DP基本思想：状态定义、状态转移、边界条件
- 一维DP经典问题：爬楼梯、打家劫舍、最长递增子序列
- 状态转移方程的设计思路
- 空间复杂度优化技巧

**重点题目：**
- [70. 爬楼梯](https://leetcode.cn/problems/climbing-stairs/)
- [198. 打家劫舍](https://leetcode.cn/problems/house-robber/)

- [746. 使用最小花费爬楼梯](https://leetcode.cn/problems/min-cost-climbing-stairs/)

## 学习建议

### 每日学习时间安排
- **上午**（1小时）：理论学习，理解算法概念
- **下午**（1.5小时）：题目练习，动手编程
- **晚上**（30分钟）：总结复习，整理笔记

### 解题步骤
1. **理解题目**：仔细阅读题目，理解输入输出
2. **分析思路**：确定使用哪种算法，设计解题步骤
3. **编写代码**：实现算法，注意边界条件
4. **测试验证**：用测试用例验证正确性
5. **优化改进**：考虑时间空间复杂度优化

### 常见错误避免
- 边界条件处理不当
- 状态转移方程错误
- 递归终止条件缺失
- 哈希表键值类型错误
- 数组越界和空指针
